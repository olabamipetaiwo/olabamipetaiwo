I research security, privacy, and trustworthy AI, with a focus on how large language models handle user data in practice versus what their privacy policies claim. My work is empirical: I build controlled experiments, measure real system behavior, and draw conclusions from evidence.

Before the PhD, I spent 4+ years as a software engineer shipping production systems across fintech, aviation, and government. That background shapes how I approach research: reproducible pipelines, rigorous evaluation, and code built to be reused.
